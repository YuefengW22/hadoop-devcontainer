# hadoop-devcontainer
A Dev Container setup for running Hadoop locally (HDFS + YARN).

Quick run (short)
1. Install Docker Desktop and VS Code (with the Dev Containers extension).
2. Open this repo in VS Code and run “Dev Containers: Rebuild and Reopen in Container”.
3. It brings up `nn`, `rm`, `w1`, `w2`, `client`.
4. Work in `/workspaces/hadoop-devcontainer` inside the container.

Access
- NameNode UI: http://localhost:9870
- ResourceManager UI: http://localhost:8088
